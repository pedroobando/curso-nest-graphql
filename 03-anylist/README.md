# DEV

1. Copiar el `.env.template` y renombrar a `.env`

2. Ejecutar

```
yarn install
```

3. Levantar (imagen docker)

```
sudo docker compose up -d
```

4. Levantar el backend nest

```
yarn start:dev
```

5. Crear la semilla con este Query

```
executeSeed
```
